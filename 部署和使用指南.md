# 题库刷题软件 - 部署和使用指南

## 系统架构

### 两个版本
1. **管理员版本**：你使用，可以导入和管理试卷
2. **用户版本**：客户使用，只能做题

### 文件结构
```
tiku_software/
├── admin/                    # 管理员工具
│   ├── activation_admin.py   # 激活码管理
│   └── paper_importer.py     # 试卷导入工具（需要密码）
├── core/                     # 核心模块
│   ├── activation.py         # 激活码系统
│   ├── question_bank.py      # 题目库
│   └── paper_manager.py      # 试卷管理
├── ui/                       # 用户界面
├── data/                     # 数据目录
├── main.py                   # 用户版本主程序
├── user_version_main.py      # 用户版本（简化）
└── 部署和使用指南.md         # 本文件
```

## 第一步：环境准备

### 1. 安装Python
确保安装Python 3.7或更高版本。

### 2. 安装依赖
```bash
pip install PyQt5 pycryptodome
```

## 第二步：管理员操作流程

### 1. 启动管理员工具
```bash
python admin/paper_importer.py
```

### 2. 登录
- 默认密码：`admin123`
- 首次启动后建议修改密码

### 3. 导入6套试卷
1. 点击"浏览"选择试卷JSON文件
2. 预览文件内容
3. 点击"开始导入"
4. 系统会自动导入所有题目和试卷

### 4. 管理激活码
```bash
python admin/activation_admin.py
```
- 生成不同类型激活码
- 查看激活码使用情况
- 延长有效期或撤销激活码

## 第三步：准备用户版本

### 1. 创建用户版本包
```bash
# 创建用户版本目录
mkdir tiku_user_version
cp -r core ui data tiku_user_version/
cp user_version_main.py tiku_user_version/main.py
cp requirements.txt tiku_user_version/
cp start.bat tiku_user_version/
```

### 2. 清理管理员文件
删除用户版本中不需要的文件：
```bash
cd tiku_user_version
rm -rf admin/
```

### 3. 测试用户版本
```bash
cd tiku_user_version
python main.py
```

## 第五步：客户使用流程

### 1. 安装软件
1. 解压ZIP文件
2. 安装依赖：`pip install PyQt5 pycryptodome`
3. 或直接运行`start.bat`（自动安装依赖）

### 2. 激活软件
1. 运行`main.py`或`start.bat`
2. 输入激活码
3. 点击激活

### 3. 使用软件
1. **练习模式**：随机做题
2. **试卷模式**：做完整的6套试卷
3. **测试模式**：模拟考试
4. **错题复习**：查看错题
5. **学习统计**：查看进度

## 第六步：数据管理

### 备份数据
定期备份以下文件：
- `data/activations.json` - 激活码数据
- `data/papers.json` - 试卷数据
- `data/questions.json` - 题目数据
- `data/admin_password.hash` - 管理员密码

### 恢复数据
使用管理员工具的"恢复数据"功能。

## 安全注意事项

### 1. 密码安全
- 修改默认密码`admin123`
- 定期更换密码
- 不要泄露密码

### 2. 数据安全
- 激活码数据加密存储
- 试卷数据未加密，不要包含敏感信息
- 定期备份数据

### 3. 防倒卖措施
- 激活码绑定设备
- 限制使用次数
- 可随时撤销激活码

## 故障排除

### 常见问题
1. **无法启动**：检查Python和依赖是否安装
2. **激活失败**：检查激活码是否正确、是否过期
3. **导入失败**：检查JSON文件格式
4. **界面显示异常**：检查PyQt5版本

### 日志文件
错误信息会显示在控制台，重要操作会记录在状态栏。
