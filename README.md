# 题库刷题软件

这是一个带有激活码功能的Python刷题软件，支持选择题、填空题等多种题型。

## 功能特点

### 1. 刷题功能
- 支持单选题、多选题、判断题、填空题
- 随机出题功能
- 练习模式和测试模式
- 错题本功能
- 学习统计和进度跟踪

### 2. 激活码系统
- 16位激活码格式：XXXX-XXXX-XXXX-XXXX
- 设备绑定功能，防止多设备共享
- 有效期管理
- 使用次数限制
- 激活码状态管理（有效、过期、撤销）

### 3. 管理后台
- 激活码生成和管理
- 激活码统计信息
- 延长有效期功能
- 撤销激活码功能

## 安装要求

### 系统要求
- Python 3.7+
- Windows/Linux/macOS

### 依赖安装
```bash
pip install -r requirements.txt
```

## 使用说明

### 1. 启动软件
```bash
python main.py
```

### 2. 激活软件
首次启动需要激活：
1. 购买激活码
2. 在激活对话框中输入激活码
3. 点击激活按钮

### 3. 使用管理后台
```bash
python admin/activation_admin.py
```

## 激活码系统说明

### 激活码格式
- 16位字母数字，格式：XXXX-XXXX-XXXX-XXXX
- 示例：A1B2-C3D4-E5F6-G7H8

### 安全特性
1. **设备绑定**：激活码绑定到特定设备，防止多设备共享
2. **加密存储**：激活码信息使用AES加密存储
3. **使用限制**：可设置最大使用次数
4. **有效期**：可设置激活码有效期

### 管理功能
1. **生成激活码**：可批量生成激活码
2. **查看详情**：查看激活码状态和使用情况
3. **延长有效期**：为激活码延长使用时间
4. **撤销激活码**：使激活码失效

## 题目管理

### 题目格式
```json
{
  "id": 1,
  "type": "single_choice",
  "category": "数学",
  "difficulty": "easy",
  "question": "1 + 1 等于多少？",
  "options": ["1", "2", "3", "4"],
  "answer": ["2"],
  "explanation": "基本的加法运算。"
}
```

### 题目类型
- `single_choice`: 单选题
- `multiple_choice`: 多选题
- `true_false`: 判断题
- `fill_blank`: 填空题

## 常见问题

### Q: 激活码可以在多台设备使用吗？
A: 默认情况下，激活码只能在一台设备使用。可以在生成时设置最大使用次数。

### Q: 激活码过期了怎么办？
A: 可以联系管理员延长有效期或购买新的激活码。

### Q: 如何备份题目数据？
A: 题目数据保存在`data/questions.json`文件中，可以定期备份。

### Q: 如何添加新题目？
A: 可以通过编辑`data/questions.json`文件或开发题目导入功能。
